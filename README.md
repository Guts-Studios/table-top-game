# Warslammer - Table Top Game

A Unity-based table-top game project.

## Project Info
- **Unity Version:** 6000.2.6f2
- **Project Name:** Warslammer

## Project Structure
```
Warslammer/
├── Assets/
│   ├── Animations/      # Animation clips and controllers
│   ├── Audio/           # Sound effects and music
│   ├── Materials/       # Material files
│   ├── Models/          # 3D models and meshes
│   ├── Prefabs/         # Prefab game objects
│   ├── Scenes/          # Unity scene files
│   ├── Scripts/         # C# scripts
│   ├── Settings/        # Project settings
│   ├── Sprites/         # 2D sprites and textures
│   └── UI/              # UI elements and assets
├── ProjectSettings/     # Unity project configuration
└── Packages/            # Package dependencies
```

## Getting Started

1. Clone this repository
2. Open the `Warslammer` folder in Unity Hub
3. Ensure you have Unity 6000.2.6f2 or compatible version installed
4. Open the project and start developing!

## Development

This project uses the Unity Input System for handling player input. The input actions are configured in `Assets/InputSystem_Actions.inputactions`.

## Contributing

Please ensure you don't commit Library, Temp, or other generated Unity files. The .gitignore is configured to handle this automatically.
